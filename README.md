# 🧠 QuizCraft — React + Tailwind Quiz App

Welcome to **QuizCraft**, a fast, modern, and fun quiz app built with **React** and styled with **Tailwind CSS**.  
Challenge yourself, track your score, and keep the momentum going with a smooth quiz flow and a clean UI.

> ✅ Built for learning, speed, and an awesome user experience.

---

## ✨ Features
- 🎮 Engaging quiz flow (questions → answers → score)
- 📊 Instant feedback + final results
- ⚡ Fully responsive UI with Tailwind
- 🎨 Clean, modern design and great readability
- 🔁 Easy restart / retake experience
- 🧩 Modular code structure for easy expansion
